# Analisis empresa de retail
EL siguiente repositorio tiene como objetivo mostrar el análisis de una empresa de retail utilizando una base de datos proporcionada por Kaggle. Primeramente se hizo el análisis exploratorio con ayuda de la librería Pandas, agregando las tablas como dataframes, para posteriormente hacer una limpieza de los datos y sacar algunas conclusiones que puedan servir como propuesta de negocio.
